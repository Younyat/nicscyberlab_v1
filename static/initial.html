Initial Setup Guide

This module handles the initial configuration and setup of the OpenStack infrastructure for the NicsCyberLab platform. Follow these steps to prepare your environment for scenario deployment.

  • Validate OpenStack credentials and environment
  • Create required network and security groups
  • Upload VM images and create keypairs
  • Configure OpenRC environment settings
  • Validate infrastructure readiness
